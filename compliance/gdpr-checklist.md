# GDPR / 个保法 合规检查清单 (GDPR Checklist)

> 适用于处理个人数据的企业应用。中国企业同时参考《个人信息保护法》。

## 一、知情同意与授权
- [ ] 在收集个人数据前，是否以清晰、易懂的方式向用户展示隐私政策？
- [ ] 用户是否能主动选择同意（Opt-in），而非默认勾选？
- [ ] 不同用途的数据收集是否分别获取独立授权（禁止"一揽子同意"）？
- [ ] 用户是否可以随时撤回同意？撤回后系统是否停止相关数据处理？

## 二、数据主体权利 (DSAR)
- [ ] **访问权**：用户能否查阅系统中存储的关于自己的全部数据？
- [ ] **更正权**：用户能否请求更正不准确的个人数据？
- [ ] **删除权（被遗忘权）**：用户能否请求删除其个人数据？删除是否覆盖全部副本（含备份系统）？
- [ ] **数据可携带权**：系统能否以通用机器可读格式（如 JSON/CSV）导出用户数据？
- [ ] **限制处理权**：用户能否请求标记数据为"限制处理状态"？
- [ ] 上述请求是否在法定时限内响应（GDPR：30 天内）？

## 三、数据最小化
- [ ] 是否仅收集业务必需的最少数据字段？
- [ ] 数据表中是否存在已无业务用途但仍在收集的字段？
- [ ] 日志和分析系统中是否避免采集非必要的个人信息？

## 四、数据存储与安全
- [ ] 个人数据是否加密存储（AES-256 或等效）？
- [ ] 传输通道是否全程 TLS 加密？
- [ ] 数据库访问是否有严格的权限控制和审计日志？
- [ ] 是否定义了数据保留期限？过期数据是否自动清理？

## 五、跨境数据传输
- [ ] 个人数据是否需要跨境传输？
- [ ] 如需跨境，是否进行了数据保护影响评估 (DPIA)？
- [ ] 是否签署了标准合同条款 (SCC) 或通过其他合法机制？

## 六、数据泄露响应
- [ ] 是否建立了数据泄露事件响应流程？
- [ ] 泄露发现后是否能在 72 小时内通知监管机构（GDPR 要求）？
- [ ] 是否能通知受影响的用户？

## 七、第三方数据处理
- [ ] 与第三方数据处理者是否签署了数据处理协议 (DPA)？
- [ ] 第三方的安全措施是否经过评估？
